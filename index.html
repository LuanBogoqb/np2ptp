What it fixes

NAT & connectivity

Too many peers can't accept inbound connections. NP2PTP falls back to a public relay automatically, so there's no port forwarding and nothing to configure.

Permanence & incentives

Content dies when seeders leave, and seeding earns nothing. Erasure coding keeps content alive through churn, as long as at least one peer already re-shared it. A reputation ledger cuts off free-riders and rewards people for seeding. That ledger is portable, too: a peer can carry a signed receipt to prove itself on first contact with a stranger, while an equally new peer with no receipt still gets choked.

Integrity & dedup

Coarse verification and no cross-content deduplication. Every chunk is content-addressed and hash-verified, so a lying peer gets caught immediately.

How it works

Content addressing

Every file is split into content-defined chunks and hashed with BLAKE3 into a Merkle tree. The root of that tree is the content's identity. A link like np2ptp:e0cf... is all you need to fetch and verify it.

Deduplication

Because chunk boundaries are defined by content, not fixed offsets, an edited file mostly reuses its old chunks. Identical files anywhere, even across unrelated downloads, get stored and transferred exactly once.

Permanence

RaptorQ erasure coding turns a file into redundant symbols, so any big-enough subset of them rebuilds the whole thing. Content survives seeders leaving, as long as at least one peer already re-shared it — nobody has to re-upload it whole.

Reachability

Built on libp2p/QUIC with DHT discovery. Behind CGNAT or a closed router? NP2PTP tries UPnP, then NAT-PMP, then falls back to a public relay automatically.

Torrent bridge

Already have a torrent's data on disk? np2ptp torrent converts it in place, verifying piece hashes straight from disk so even a 50 GB torrent never touches RAM. Once it's bridged, any NP2PTP peer can find it from then on, not just whoever had the original torrent.

Three commands

Link a file, seed it, fetch it. That's the whole workflow.

Link what you want to share
np2ptp pack myfile.zip --out myfile.nptp
Make it available on the network
np2ptp serve myfile.nptp
Download it, anywhere
np2ptp fetch np2ptp:abc123... --out ./downloaded
